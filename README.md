## 🛠️ How to Use?

1. Clone the repository:
   ```sh
   git clone https://github.com/YOUR_GITHUB/YOUR_REPOSITORY.git
   ```
2. Navigate to the project folder:
   ```sh
   cd YOUR_REPOSITORY
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the project:
   ```sh
   npm start
   ```
